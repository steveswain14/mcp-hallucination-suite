What's included

Four suppressors. One endpoint.

grounding_enforcer
Catches ungrounded claims, fabricated citations, and attributions that can't be matched to a real source.
prompt_suppressor
Detects prompt injection attempts embedded in retrieved content, user inputs, or tool results.
json_suppressor
Validates tool call responses against expected schemas. Rejects malformed, truncated, or injected JSON.
tool_response_suppressor
Validates tool responses against declared outputs. Flags unexpected fields and schema drift.
Integration

One POST request. No config.

// Call before your agent output reaches users

const res = await fetch('https://mcp-hallucination-suite-production.up.railway.app/validate', {
  method: 'POST',
  headers: {
    'X-API-Key': 'your-api-key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    agent_turn: { grounding: { model_output: output, retrieved_sources: sources } },
    run: ['grounding', 'prompt', 'json']
  })
});

const { total_violations } = await res.json();
// total_violations === 0 ? ship it : handle it
